## Это репозиторий проекта "Менеджер задач"

Приложение **умеет**:
1. Принимать задачи с фронтенда
2. Выводить задачи списком
3. Получать определенную задачу по индексу
4. Удалять по индексу
5. Удалять все задачи
6. Получать список всех подзадач определенной задачи
7. Обновлять задачи
8. Выводить историю просмотренных задач

Приложение написано на Java. Пример кода:
```java
public class Practicum {
    public static void main(String[] args) {
    }
}
```
